//How to setup project //
// //
///////////////////////////////////

step 1:Unzip the workspace from the folder :/Apollo/project

Example : g:/Apollo/project

step 2:Open the WSAD change the workspace to project which is given in step 1

Step 3: Execute this query
"restore database d1 from e:/ into d1 without prompting"
by default all schema is pointing db2 only unless otherwise mentioned in the query.

To create a db2 schema: create a useraccount named db2admin with schema IPSIT

step 4: i)Run the file Apollo/WebContent/front.jsp on server
ii)Choose the Test Environment from the list of servers.
iii)Port number should be 8080
iv)Click ok.
